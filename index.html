Core Modules

üìã Ledger

Immutable audit trail with SHA-256 hashing. Every action logged with timestamp, user, and cryptographic integrity.

const ledger = new Ledger();
ledger.record({
  action: 'PROJECT_CREATED',
  projectId: 'CPF-001',
  userId: 'john@org.com'
});
ledger.verify(); // Check integrity

üîê RBAC

Role-based access control with 7 predefined roles: PM, Finance, QA, Executive, Auditor, Contractor, Customer.

const rbac = new RBAC();
rbac.defineRole('pm', {
  view: '*',
  approve: ['assessment', 'bids']
});
rbac.can('john', 'approve', 'payment');

‚úÖ Approval Engine

Multi-party milestone approvals with configurable chains and threshold-based routing.

approvals.requestApproval({
  projectId: 'CPF-001',
  milestone: 'payment',
  amount: 15000,
  requiredApprovers: ['finance', 'exec']
});

üí∞ Funding Tracker

Multi-source funding commitments with real-time gap analysis and disbursement tracking.

funding.addCommitment({
  projectId: 'CPF-001',
  source: 'Energy Trust',
  amount: 10000
});
funding.getGap('CPF-001');

üéØ QA Gates

Quality checkpoints with pass/fail gates preventing progression without approval.

qa.create({
  projectId: 'CPF-001',
  stage: 'hea',
  inspector: 'qa_tom'
});
qa.pass('CPF-001', 'hea');

üì¶ Export & Backup

ledger.verify(); // Integrity check
